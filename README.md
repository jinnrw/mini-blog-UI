# mini-blog-ui
A mini blog UI built with React and Redux.

## Built With
- React, create-react-app
- Redux, react-redux
- TailwindCSS
- Testing
  - Jest, Enzyme (in progess)

### Demo
Visit the [demo](https://mini-blog-ui.vercel.app/) here! (Deployed with Vercel)

## Run Locally
```bash
npm run start
# or
yarn start
```
